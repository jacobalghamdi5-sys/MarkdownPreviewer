#  Jacob Corp. | MarkdownPreviewer

An advanced, enterprise-grade Markdown IDE designed for clean code block syntax highlighting, real-time metrics, and instant browser-side rendering.

##  Repository Architecture
* **`.github/workflows/`** - Automated CI/CD deployment pipelines.
* **`index.html`** - Core UI workspace and execution engine.
* **`.gitignore`** - System environment safety exclusions.

##  Core Features
-  **Dynamic Local Storage Autosave** - Progress is securely cached on every single keystroke.
-  **Synchronized Scrolling** - The compiler view smoothly mirrors the source input tracking path.
-  **Real-time Analytics** - Live tracking for word counts, characters, and estimated reading times.
-  **GitHub Alert Parsing** - Fully supports custom syntax like `> [!WARNING]` out of the box.
